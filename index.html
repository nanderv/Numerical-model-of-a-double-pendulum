Numerical-model-of-a-double-pendulum by nanderv

A program to display the result of a numerical approximation of a pendulum with a rotating suspension point.

Dependencies

SFML GCC

Install

Install any gcc compiler (Mingw for windows) Download and install sfml Download and install code::Blocks Open the codeblocks project Change the settings to the paths on your machine Compile Run
